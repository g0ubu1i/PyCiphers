# RSA

---

## 密钥生成

1.选择两个大素数`p`和`q`

2.计算模数$n = p*q$

3.计算欧拉函数$\phi(n)= (p-1)*(q-1)$

4.选择指数`e`，要求$1<e<\phi(n)$,且$gcd(e,\phi(n))\ =1$

5.计算私钥指数`d`,满足 `d ≡ e⁻¹ mod φ(n)`

得到：

- （e,n）公钥，用于加密
- （d,n）私钥，用于解密

## 加密过程

将m转换为整数，要求$0<m<n$,然后计算密文

$$
c = (m^e) \mod n
$$

## 解密过程

将密文c解密得到明文

$$
m = (c^d) \mod n
$$
